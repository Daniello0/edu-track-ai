# UI/UX Design Specification: EduTrack AI

> **Источник истины по коду:** [.cursor/rules/project-conventions.mdc](../.cursor/rules/project-conventions.mdc)  
> **Продукт и архитектура:** [main-design.md](./main-design.md)  
> **Схемы компонентов и состояния:** [schemas-design.md](./schemas-design.md) §4–5

## 1. Визуальная концепция

**Стиль:** Soft Minimalism с элементами Glassmorphism.  
**Атмосфера:** «Спокойное обучение» — мягкие пастельные тона, скруглённые углы, лёгкое размытие фона (`backdrop-filter`).  
**Принцип интерфейса:** минимум когнитивной нагрузки; крупные элементы управления, щедрые отступы, фокус на тексте.

**Радиусы скругления:**
- Карточки и инпуты: `16px`
- Модальные окна и крупные панели: `24px`

## 3. Структура страниц

### 3.1. Header (общий компонент)

- **Слева:** логотип (иконка книги + «EduTrack AI»).
- **Справа:**
  - Переключатель темы (`Sun` / `Moon` из Lucide React).
  - Гость: кнопка «Войти» (outline).
  - Авторизованный пользователь: иконка `User` с выпадающим меню (Профиль, Выйти).

### 3.2. Главная страница (`main-page`)

- **Фон:** одно размытое органическое пятно пастельного цвета с медленной CSS-анимацией смещения.
- **Центральный блок:**
  - **Поле ввода:** широкий input, мягкая тень, placeholder: «Вставьте ссылку на YouTube видео…».
  - **Блок настроек** (Segmented Control + селекты):
    - Формат: «Рассказ» / «Саммари».
    - Объём саммари (виден только при «Саммари»): «Краткий» / «Средний» / «Полный».
    - Язык: выпадающий список — RU, EN, Оригинал.
    - Тестирование: toggle Вкл/Выкл; при включении — счётчик вопросов (1–10) и вариантов ответа (3–5).
  - **Кнопка:** «Читать» (primary), при наведении — `transform: scale(1.02)`.

### 3.3. Режим чтения (`reader`)

- **Фон:** однотонный кремовый `#FFFDF5` (light) / `#22232A` (dark).
- **Контент:** текст по центру, `max-width: 800px`, шрифт `Lora`.
- **Шапка:** заголовок видео, бейдж категории, дата обработки.
- **Нижняя панель** (sticky):
  - Тест выключен: «Сохранить» (primary), «Удалить» (secondary).
  - Тест включён: «Перейти к тестам» (primary).

### 3.4. Экран тестирования (`quiz`)

- **Layout:** один вопрос на экран, навигация «Назад» / «Далее».
- **Прогресс:** индикатор «Вопрос X из Y».
- **Взаимодействие:** при выборе варианта — мягкая подсветка `rgba(178, 226, 212, 0.4)`.
- **Результат:** процент успеха, список ошибок с правильными ответами.
- **Финал:** «Сохранить в библиотеку» (primary), «Удалить и выйти» (secondary outline).

### 3.5. Профиль (`profile`)

- **Дашборд:** три плитки статистики — «Изучено минут», «Пройдено тестов», «Любимая категория».
- **Список материалов:** карточки с превью YouTube, заголовком, датой `last_viewed_at` и бейджем статуса.
- **Действие на карточке:** кнопка «Продолжить» → экран чтения.

## 4. Специфические механики

### Сохранение для гостя

1. Гость нажимает «Сохранить».
2. Модальное окно: «Чтобы не потерять конспект, создайте аккаунт».
3. После регистрации/входа — автоматический `POST /api/library` с данными из `sessionStorage`.

### Состояния загрузки

| Шаг | Отображение |
| :--- | :--- |
| `transcribing` | Пульсирующая иконка «AI» + текст «Получаем транскрипт…» |
| `ai_processing` | Тот же индикатор + текст «Обрабатываем материал…» |
| `completed` | Плавный fade-in экрана чтения |
| `error` | Toast с описанием ошибки, кнопка «Попробовать снова» |

### Переходы между экранами

Все смены экранов — `fade-in` 300ms (`opacity` 0 → 1).

## 5. Адаптивность

- **Breakpoint:** `768px`.
- **Мобильная вёрстка:** настройки располагаются под полем ввода (вертикальный стек).
- **Чтение:** `max-width: 100%`, боковые отступы `16px`.
- **Дашборд:** плитки статистики в одну колонку на мобильных.

## 6. Техническая реализация UI
